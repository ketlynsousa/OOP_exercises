# Python OOP Exercises – Curso em Vídeo

This repository contains the exercises, experiments, and challenges I completed while studying **Object-Oriented Programming (OOP) in Python** through the **Curso em Vídeo** series, taught by Professor Gustavo Guanabara.

## 📚 About the Course

This course focuses on **Object-Oriented Programming with Python**, introducing fundamental concepts used in modern software development.

Key topics include:

* Classes and Objects
* Attributes and Methods
* Instance vs Class Attributes
* Encapsulation and Object State
* Practical object modeling with Python

The lessons combine theory with practical exercises, encouraging students to model real-world concepts using object-oriented design.
